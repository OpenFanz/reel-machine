# Reel Machine (Free) — clone any viral AI reel with YOUR own model

Take a reel that's **already going viral** and rebuild it with **your own consistent AI model** — same face,
same body, every time. **No install. No code. No Python.** Just Claude Desktop + your browser.

## How it works — 2 skills (run in order)
1. **reel-intake** — you drop a few screenshots of the viral reel → Claude figures out exactly how it's built.
2. **seedance-prompter** — Claude writes the full Seedance 2.0 prompt from the reel's format + your model → you paste it into WaveSpeed, upload your reference images, generate. Out comes your reel.

**You bring your model's reference images.** Building a consistent model from scratch is the `_skool/character-builder` bonus (the "owned asset" part).

## The one rule that makes it work
**Identity comes ONLY from reference images — never from words.** You never describe your model's face/hair/body
in a prompt (that causes "drift"). You describe the *outfit* and the *scene*; her identity is carried by the images.

## Setup (2 minutes, nothing to install)
1. A **WaveSpeed** account → https://wavespeed.ai (pay-per-generation, no subscription). One account does BOTH the character images AND the videos.
2. Add these 3 skills to **Claude Desktop** (Settings → Skills → add this folder), **or** just make a Claude
   Project and drop these files in as knowledge. Then talk to Claude normally.

## What it costs (on WaveSpeed, pay per generation)
- Character images (Nano Banana Pro): ~$0.14 each.
- One reel (Seedance 2.0): ~$0.60 (draft cheap → finalize once).

## The flow
```
[setup]    have your model's reference images ready (4–6 consistent shots)
[per reel] "clone this reel"      + a few screenshots  → reel-intake → seedance-prompter → your reel
```

## Want it fully automated?
This free version has you paste each prompt into WaveSpeed yourself. There's also a **fully-automated version**
where Claude fires every generation itself and pumps out reels on autopilot — that's inside the Skool.

> Only clone the FORMAT. Don't republish someone's file, copy their face/handle/exact lines, or use audio you
> don't have rights to. Rebuild the structure with your own model.
